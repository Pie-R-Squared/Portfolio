Platformer Game

During my A-levels I developed a side-scrolling platformer game written in Python, making extensive use of the pygame library. The project took about a year and a half to complete and detailed documentation (consisting of 187 pages) was made alongside it.

The main features of the game include:

  • Interactive main menu
  • Controls screen
  • Character select screen
  • Main game window
  • A high-score system, where the highest score is saved to an external file and displayed on the gameover screen
  • Confirmation dialog when attempting to press esc during gameplay.

GameGuardian Script

GameGuardian is an android game hacking application, requiring root access to utilise. More specifically, it's a memory editor, capable of editing values in RAM of running processes.

The scripts are written in lua and executed inside GameGuardian with the app running in the background. This script that I've written is a simple menu which includes a sub-menu, with features such as spawning animals and resizing the player character in real-time. Developing this script took a few days and was an enjoyable experience to test and build upon. Similar more complicated scripts can also be made which use offset addressing instead of searching and refining values.

This particular script is accessible on my GitHub page, which is linked beneath the code thumbnail. Currently, only two versions have been created as the features are quite basic, with limited functionality on it's own; I may later incorporate it into a larger menu.

Other features I have implemented in newer scripts include:

  • Cloning the player
  • Blank player name
  • Immortality
  • Infinity points
  • Boss spawn
